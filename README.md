Automation with Playwright of entering a date in "https://testautomationpractice.blogspot.com/".

Project goal - to test date field and entry in test automation practise website in TestNG framework.

Project was created using IntelliJ, Maven Build System, Java and Playwright.

Project consists of:

1. PlaywrightUtils class;

2. Date Section class;

3. Date Tests class.

4. Extent Reports.
